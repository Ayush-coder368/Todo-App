# 📋 iTask – Your Task Manager

A clean and minimal **To-Do Web Application** built using **HTML, CSS, and JavaScript**.
iTask helps you organize your daily tasks efficiently with a simple and intuitive interface.

---
live working link - https://todo-app-five-beryl-16.vercel.app/

## 🚀 Features

Based on the application UI:

* 📝 **Task Management**

  * Add new tasks easily using the **"Add New Task"** button.
  * Organize tasks in a structured workflow.

* 📂 **Three Task Stages**

  * **To Do** – Tasks that are yet to be started.
  * **In Progress** – Tasks currently being worked on.
  * **Completed** – Finished tasks.

* 🔢 **Task Counters**

  * Each section displays the number of tasks it contains.

* 🎯 **Simple Workflow**

  * Move tasks between columns to track progress visually.

* 🌙 **Dark Mode UI**

  * Clean, modern dark-themed interface for better usability and reduced eye strain.

* 💾 **Local Storage (LocalDB)**

  * All tasks are stored in the browser’s **local storage**.
  * No backend required.
  * Your data persists even after refreshing the page.
* **Drag-and-drop functionality**
* **Delete task options**

---

## 🛠️ Tech Stack

* **HTML** – Structure of the app
* **CSS** – Styling and layout (Dark UI)
* **JavaScript** – Functionality and interactivity
* **Local Storage** – Data persistence in the browser

---

## 🧠 How It Works

* Tasks are created via the UI.
* Each task is stored in **Local Storage** as JSON.
* The app dynamically updates:

  * Task lists
  * Task counts
* On reload, tasks are fetched from local storage and restored.

---

## 📸 UI Overview

The app consists of three main panels:

* Left: **To Do**
* Center: **In Progress**
* Right: **Completed**

Each panel is designed for clarity and smooth task tracking.
